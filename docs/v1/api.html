      /users/{userUuid}/account/transfer post

      post

      Realiza uma transferência para uma outra conta

      post /users/{userUuid}/account/transfer

      Realiza uma transferência para uma outra conta

      Secured by basicAuth

      Todas as requisições precisam conter as credencias de autenticação válidas

      • Request
      • Response
      • Security

      URI Parameters

      • userUuid: required (string)

      Body

      Media type: application/x-www-form-urlencoded

      Type: object

      Properties
      • account_number: required (string)

        Número da conta a ser transferido

      • balance_amount: required (number)

        Valor em R$ a ser descontado do saldo da conta bancária do usuário que está realizando a transferência

      • credit_card_amount: required (number)

        Valor em R$ pago com cartão de crédito para completar a transferência, somente quando a conta não tem saldo suficiente

      • total_amount: required (number)

        Valor em R$ representando o total a ser transferido

      • payment_method: required (string)

        Método de pagamento opções disponíveis: balance, credit_card, balance_and_cc

      • card_id: (string)

        ID do cartão do crédito utilizado para completar o valor da transacao

      HTTP status code 200

      Body

      Type: any
